## Deployment

### Opzione 1: GitHub Pages (Gratuito)

1. Crea un repository su GitHub
2. Carica i file del sito
3. Abilita GitHub Pages nelle impostazioni
4. Il sito sarà disponibile su `https://username.github.io/chioggia-bus-website`

### Opzione 2: Netlify (Gratuito)

1. Vai su [netlify.com](https://netlify.com)
2. Collega il repository GitHub
3. Netlify farà il deploy automatico
4. Dominio personalizzato disponibile

### Opzione 3: Vercel (Gratuito)

1. Vai su [vercel.com](https://vercel.com)
2. Importa il progetto da GitHub
3. Vercel farà il deploy automatico
4. Dominio personalizzato disponibile

### Opzione 4: Server Personalizzato

```bash
# Avvia il server locale
python3 -m http.server 8000

# O con Node.js
npx http-server
```

## Configurazione Dominio Personalizzato

Per usare un dominio personalizzato (es. `bus-chioggia.it`):

1. Registra il dominio presso un registrar (GoDaddy, Namecheap, ecc.)
2. Configura i DNS per puntare al servizio di hosting
3. Abilita HTTPS nel pannello di controllo
